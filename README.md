# Vivum - пиксельная песочница с различными частицами

Это браузерная симуляция сыпучих тел в оболочке старой версии Windows. Сыпь песок, лей воду и пылай огнём!

---

## Как это работает

1. Запускается загрузочный экран с анимированной полосой
2. Появляется рабочий стол с пиксельными обоями
3. Двойной клик по иконке Vivum открывает окно симуляции
4. Выбираешь элемент на панели инструментов и рисуешь мышью
5. Частицы падают, взаимодействуют и накапливаются по физике

---

## Функции

- **Загрузочный экран** - анимация в стиле Windows XP с анимированной полосой
- **Рабочий стол** - пиксельные обои, иконки, таскбар, часы, трей
- **Меню Пуск** - открытие игры, пресеты, быстрое сохранение, гитхаб
- **Физика частиц** - песок, вода, огонь, растения, лава, кислота, масло и др.
- **Пресеты** - готовые сцены: вулкан, ледник, оазис, каскад
- **Быстрое сохранение** - сохранение и восстановление состояния холста
- **Регулировка размера кисти** - можно выбирать размеры кисти
- **Звуки** - звуки взаимодействия с системой, а также в самой игре

---

## Стек

### Интерфейс
- HTML / CSS / JavaScript - без фреймворков
- CSS-анимации - загрузочный экран, fade-in иконки, полоска
- SVG - иконки, иконка ОС, элементы UI

### Физика
- WebAssembly - движок симуляции частиц
- Canvas API - рендеринг симуляции в реальном времени

### Графика
- `<canvas>` + `ImageData` - пиксельный рендер обоев
- Квантизация цвета - 16-цветная палитра в стиле игры

---

## Запуск

### Требования
- Любой современный браузер с поддержкой WebAssembly
- Локальный веб-сервер (из-за ограничений WASM на `file://`)

### Локально

```
git clone https://github.com/Joehawkk/vivum.git
cd vivum
```

Запусти любым сервером, например:

```
npx serve .
```

Открой `http://localhost:3000` в браузере.

## Элементы

| Элемент   | Поведение                              |
|-----------|----------------------------------------|
| Sand      | Сыпется, накапливается горкой          |
| Water     | Течёт, заполняет пространство          |
| Fire      | Горит, поджигает некоторые элементы    |
| Plant     | Растёт при контакте с водой            |
| Lava      | Горячая жидкость, поджигает и твердеет |
| Ice       | Замораживает воду, тает от огня        |
| Acid      | Разъедает большинство элементов        |
| Oil       | Горючая жидкость, легче воды           |
| Stone     | Твёрдый элемент                        |
| Rocket    | Летит вверх, копируя элементы          |

## Структура

```
vivum/
├── index.html              # Точка входа - весь ОС-интерфейс
├── assets/
│   └── bliss.jpg           # Фото для пиксельных обоев
├── css/
│   ├── style.css           # Основные стили ОС-интерфейса
│   └── xp-override.css     # Переопределение стилей движка
├── js/
│   ├── boot.js             # Загрузочный экран и открытие окна
│   ├── vivum-chrome.js     # Окно, таскбар, меню Пуск, часы
│   ├── vivum-ui.js         # Панель элементов и кисти
│   ├── wallpaper.js        # Пиксельные обои через Canvas
│   ├── presets.js          # Пресеты сцен
│   ├── sounds.js           # Звуки ОС
│   ├── elements.js         # Таблица элементов и цветов
│   ├── renderer.js         # Canvas-рендер частиц
│   ├── simulation.js       # JS-порт физики частиц
│   └── dialog.js           # ОС-диалог подтверждения
└── engine/
    ├── loader.js           # Webpack-загрузчик движка
    ├── core.js             # Ядро симуляции
    ├── ui.js               # UI движка
    └── vivum.module.wasm   # Движок симуляции частиц
```
